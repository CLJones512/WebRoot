Helpful articles on Android sensors


  1. Pixel 8 Pro Temperature Sensor - The Verge

    This is a Verge article about a new temperature sensor on the Google Pixel 8 Pro. I found that
    while the information about this fature was really cool, it's probably not super useful within
    the scope of what I'm doing for this project. First off, since this is still a relatively new
    feature, I wouldn't necessarily want to develop an app using this because it's not on a lot of phones.
    Also, judging by the article, it's not super reliable quite yet.

    Usefulness: 2 out of 5
    Link

  2. Sensors Overview - Android Developers

    This resource will likely prove far more useful. It's official documentation from Google about the
    sensors on Android phones and how to make use of them in your development work. It provides a very
    comprehensive list of all the sensors, what the sensors do, and how to use them in your programming.

    Usefulness: 5 out of 5
    Link

  3. All the Sensors in Your Smartphone and How They Work - Gizmodo

    This is a Gizmodo article giving a rough overview of how all the sensors work in your phone. This will
    probably be the first article I really dive into, so that I have a rough overview of how these sensors
    work before I dive into the official documentation so that I'm not completely flustered by all the
    information there. While it'll definitely make good surface level research, I don't think it'll be
    something I reference when I make my deep dive into programming one of these sensors, so that's the only
    thing stopping me from giving this a higher score

    Usefulness: 4 out of 5
    Link

  4. Android - Sensors - TutorialsPoint

    This is a resourse that provides tutorials for how to program various Android Sensors. I feel like this
    prove to be more useful than the official documentation, since past experiences have proven that checking
    the official documentation when I'm just starting out can prove a bit confusing, since most places assume
    you have a fair amount of knowledge in the subject already. This presents the information in a much more
    beginner friendly way, so I'll absolutely check this one a lot when I'm starting out.

    Usefulness: 5 out of 5
    Link

  5. w3schools

    Last, but certainly not least, is w3schools. Ever since I started programming I've used this, and while it might
    not have anything directly relating to sensors, it certainly includes a lot of resources relating to concepts
    like HTML, JavaScript, React, Kotlin, Node.js, and other tools that we'll be using in this class. So it's definitely
    going to be a resource I use a lot this quarter

    Usefulness: 5 out of 5
